# Endless Line — Спецификация проекта

## Бренд
- **Название:** Endless Line (не переводится, одинаковое на всех языках)
- **Логотип:** Утверждён. Mallard A4 с вагонами, паром, оранжево-белый круглый эмблема
- **Резервная копия логотипа:** `assets/backup/logo_approved_v1.png`
- **Цветовая палитра:** Оранжевый (#FF6B35) + Белый + Тёмно-серый

## Технический стек
- HTML5 / CSS3 / Vanilla JS
- Шрифты: Outfit (заголовки), Inter (текст)
- Без фреймворков
- Сервер: http-server (порт 8080)
